## Requirements
### Requirement: Favicon Uses App Icon
The system SHALL set the browser favicon to the application icon across the web UI.

#### Scenario: Base layout renders favicon
- **WHEN** a user loads any page using the base layout
- **THEN** the favicon references the application icon asset
