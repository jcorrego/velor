# Change: Add Integration Connection Management

## Why
Integrations need a shared foundation for connections, credentials, and sync tracking.

## What Changes
- Create integration connection records with statuses and metadata.
- Store credentials securely and track sync history.
- Provide a UI to connect, disconnect, and refresh integrations.

## Impact
- Affected specs: integration-management
- Affected code: integration connections, credentials storage, sync logging, UI
