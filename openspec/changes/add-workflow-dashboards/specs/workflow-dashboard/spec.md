## ADDED Requirements
### Requirement: Guided Workflow
The system SHALL provide a guided workflow per jurisdiction and tax year.

#### Scenario: View workflow steps
- **WHEN** a user opens a jurisdiction workflow
- **THEN** the system SHALL display the current step and remaining steps

### Requirement: Dashboard Overview
The system SHALL provide a dashboard with cross-jurisdiction status and alerts.

#### Scenario: View dashboard overview
- **WHEN** a user visits the dashboard
- **THEN** the system SHALL display summary metrics and alert counts
