# Change: Add US Form 1120 Reporting

## Why
US corporate filers need a Form 1120 summary that works like the existing US tax reporting flows.

## What Changes
- Add Form 1120 summary reporting using category tax mappings and USD normalization.
- Add a Form 1120 reporting UI that follows the same filing-selection pattern as Schedule E and Form 5472.

## Impact
- Affected specs: us-tax-reporting
- Affected code: US tax reporting service, UI pages, routes, tests
