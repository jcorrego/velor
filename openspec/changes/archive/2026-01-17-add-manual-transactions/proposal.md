# Change: Add Manual Transaction Entry and Editing

## Why
Users need to add and correct transactions that are missing from imports or require quick adjustments without leaving the Finance UI.

## What Changes
- Add a manual transaction form for creating transactions in the Finance UI.
- Add an "Add transaction" button to open the manual transaction form.
- Allow editing existing transactions from the Transactions list.
- Validate required fields and enforce account ownership.

## Impact
- Affected specs: finance-management
- Affected code: Livewire finance components, transaction validation, transaction UI
